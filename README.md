# Intelligent Intrusion Detection System

## Description
This project implements an Intelligent Intrusion Detection System (IIDS) using Python to enhance network security through real-time monitoring and automated threat detection.

## Features
- Real-time monitoring of network traffic.
- Machine learning algorithms for threat detection.
- User-friendly interface for easy interaction.
- Log analysis and reporting features.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/VISHWANATHAN13/Intelligent-intrusion-detection-system-in-python.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Intelligent-intrusion-detection-system-in-python
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the Intrusion Detection System, use the following command:

```bash
python main.py
```

## Run Commands
- Starting the system:
  ```bash
  python main.py
  ```
- Checking logs:
  ```bash
  cat logs/system.log
  ```
